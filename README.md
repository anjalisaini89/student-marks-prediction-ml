# 📚 Student Marks Prediction using Linear Regression

A beginner-friendly Machine Learning project built using **Python** and **Scikit-learn** to predict student marks based on study hours.

---

## 🚀 Features

- Data preprocessing using Pandas
- Linear Regression Model
- Correlation Analysis
- Statistical Summary
- Actual vs Predicted Comparison
- Performance Metrics
  - MAE
  - MSE
  - RMSE
  - R² Score
- Regression Line Visualization
- User Input Prediction

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/yourusername/student-marks-prediction.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python student_marks_prediction.py
```
